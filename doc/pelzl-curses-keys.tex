\begin{Name}{1}{pelzl-curses-keys}{Paul J. Pelzl}{a configuration utility for pelzlrc}{pelzl-curses-keys manpage}
   \Prog{pelzl-curses-keys} is a small utility designed to assist in the creation of an \Cmd{pelzlrc}{5} file.
\end{Name}

\section{Synopsis}
\Prog{pelzl-curses-keys}

\section{Description}
\Prog{pelzl-curses-keys} is a small interactive utility designed to help users of the \Cmd{pelzl}{1} calculator
to create a customized \Cmd{pelzlrc}{5} file.  \Prog{pelzl-curses-keys} translates keypresses into octal and
string representations that can be used with the 'bind' command inside pelzlrc.  Ctrl-C will exit the program.
