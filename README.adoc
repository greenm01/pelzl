Pelzl
=====

Pelzl is a terminal calculator designed for precision and speed. It provides a classic RPN interface by default, while offering a robust algebraic REPL for those who prefer a modern expression-based workflow.

This project is an evolution of the Orpie calculator and is dedicated to the memory of its original author, https://web.archive.org/web/20230529194200/https://www.bradshawfuneral.com/obituaries/paul-pelzl[Paul Pelzl].

== Features

* **Classic RPN UI**: A traditional stack-based workflow, right out of the box.
* **Algebraic REPL**: Evaluate expressions naturally: `(5 + 3) * 2`. Launch with `--repl`.
* **Implicit Multiplication**: Support for concise notation, such as `5sin(30)` or `2x`.
* **Variable Assignment**: Persist values with simple declarations: `x = 42`.
* **Mathematical Functions**: Comprehensive support for `sin`, `cos`, `sqrt`, `ln`, and more.
* **Mosaic TUI**: A responsive, colorful interface built on the Mosaic library.

== Installation

=== Using OPAM

Install Pelzl directly via https://opam.ocaml.org/[OPAM]:

[source,shell]
----
opam install pelzl
----

=== Manual Installation

To build from source, you will need OCaml (4.03 or newer) and the following packages: `dune`, `mosaic`, `gsl`, `num`, `str`, and `camlp-streams`.

[source,shell]
----
# Build the project
make

# Install (may require sudo)
make install
----

== Usage

Launch the classic RPN interface by running `pelzl`. To use the algebraic REPL, run:

[source,shell]
----
pelzl --repl
----

Consult the `doc` directory for further details.
